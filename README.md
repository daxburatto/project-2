# E-commerce Back End Starter Code

# Installation
  * Open the terminal
  * Type `npm init --y` Hit enter
  * Type `npm install --save mysql2` Hit enter
  * Type `npm install express` Hit enter
  * Type `npm install dotenv` Hit enter
  * Type `npm install sequelize` Hit enter

# Built With
* Sequelize
* JavaScript
* Dotenv
